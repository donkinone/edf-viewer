🔧 技术修改详情

📝 JavaScript修改：

  • 移除时间轴创建逻辑：删除 timeAxis 元素创建代码
  • 移除时间刻度计算：删除 timeMarks 数组和循环计算
  • 移除DOM插入：删除 waveformContainer.appendChild(timeAxis)
  • 保留时间格式化函数：formatTimeFromSeconds 仍用于其他地方
移除的JavaScript代码：

// 添加时间轴（基于当前显示的时间范围）
const timeAxis = document.createElement('div');
timeAxis.className = 'eeg-time-axis';

// 创建时间刻度（10个间隔，11个标记点）
const timeMarks = [];
const numMarks = 10;

for (let i = 0; i <= numMarks; i++) {
  const timeProgress = i / numMarks;
  const currentTime = actualStartTime + (actualDisplayDuration * timeProgress);
  timeMarks.push(formatTimeFromSeconds(currentTime));
}

timeAxis.innerHTML = timeMarks.map(mark => `<span>${mark}</span>`).join('');
waveformContainer.appendChild(timeAxis);

🎨 CSS修改：

  • 移除主要样式：.eeg-time-axis 和 .eeg-time-axis span
  • 移除定位样式：11个子元素的精确位置定义
  • 移除响应式样式：移动端的时间轴适配
  • 减少样式文件大小：删除约30行CSS代码

✅ 移除后的优势

🚀 用户体验提升：

  • 界面简洁：减少视觉元素，专注于波形数据本身
  • 空间利用：节省垂直空间，可显示更多通道
  • 减少干扰：避免时间标记与波形数据的视觉冲突
  • 一致性：时间信息统一显示在控制面板中

⚡ 技术优势：

  • 性能提升：减少DOM元素数量，提高渲染速度
  • 代码简化：移除复杂的时间轴计算和定位逻辑
  • 维护性：减少需要维护的代码量
  • 响应式：简化移动端适配逻辑
